We'll cover the following fundamental topics during this section of class:

* Classes, Subclasses, and Object Oriented programming
* Activities and Activity Lifecycle
* Views and Layouts
* Intents
* Event Listeners
* Custom Adapters
* Android Debugging
* Resource file best practices
